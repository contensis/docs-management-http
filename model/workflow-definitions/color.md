# Workflow colors

Colors which are used in the Contensis UI for workflow [states](state) and [events](event).

## Supported colours

The following list contains the colours that are currently supported:

- Red
- Green
- Orange
- Blue
- Gray
- Purple
- HotPink
- Cyan
- Lime
- Ruby
- LightGray
- Yellow
